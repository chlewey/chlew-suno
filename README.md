# Suno Lyrics Collection

A small creative workspace for song ideas, lyrics, and audio stems.

## Overview

This repository contains lyrics and music files for a collection of songs produced with Suno and related tools.

- `.suno.txt`, `.suno`, `.org`, and `.txt` files contain song lyrics, drafts, style notes, and sometimes AI-generated sections.
- `.wav`, `.mp3`, and `.m4a` files are audio recordings or stems associated with the songs.

## Project Notes

- **Director:** Carlos Thompson - *Chlewey*
- **Producer:** Suno
- Lyrics and styles are often a mix of human-written and AI-generated content.
- AI tools referenced in this repository include Suno, Claude, ChatGPT, Gemini, and Grok.

## Usage

Open the Markdown and text files to review lyrics, song structure, and production notes.

If you want to extend this collection:

1. Add new `.suno.txt` or `.suno` files for song drafts.
2. Keep audio stems and recordings organized with matching file names.
3. Note attribution clearly when using AI-generated lyrics or styles.

## Attribution

Some songs are entirely composed by Carlos Thompson, while others were co-created with AI.
Detailed attribution may appear in individual song files.

## Git Ignore

The repository currently ignores audio and large media files with:

```
*.wav
*.mp3
*.m4a
```

This keeps the version control history focused on lyrics and metadata.
